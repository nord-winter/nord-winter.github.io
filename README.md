# nord-winter.github.io

Personal portfolio site — [nord-winter.github.io](https://nord-winter.github.io)

Built with [Astro](https://astro.build/) and [Tailwind CSS v4](https://tailwindcss.com/).

## Local development

```bash
npm install
npm run dev
```

## Configuration

All content lives in `src/config.ts` — name, bio, skills, experience, education, certifications.

Photos are served from Cloudinary. Credentials go in `.env` (see `.env.example`).

## Deploy

Deployed automatically via GitHub Actions on push to `main`.
